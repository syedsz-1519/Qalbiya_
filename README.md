# Qalbiya Islamic Institute 📖✨

> **Warm, spiritually grounded, and highly interactive online Islamic education for women and children.**

Qalbiya Islamic Institute is a premium, beautifully crafted web platform built with React, TypeScript, and Tailwind CSS. The website showcases academic circles, Tarbiyah courses, and free resources designed specifically for adult sisters and children (ages 6–12) under dedicated female mentorship.

---

## 🌟 Key Features

- **Dynamic Program Hubs**: Dedicated navigation routes and cards for both **Women's Academy** and **Kids' Tarbiyah Hub**.
- **Interactive Navigation Dropdown**: Premium "Programs" hover menu on desktop and a clean mobile accordion sub-navigation drawer.
- **Detailed Course Layouts**: Fully responsive course detail pages showcasing structured syllabi, FAQs, teacher notes, and a visually prominent Outcome quote section with faded watermarks (`Akhlaq`, `Deen`, `Quran`, `Tajweed`, `Tarbiyah`).
- **Free Resources Hub**: A filterable catalog of free digital guides, live meeting schedules, and printable trackers categorized for **Sisters**, **Kids**, and **Everyone**.
- **Scholarship Pipeline**: Built-in, confidential sponsorship application form for students requiring financial accommodation.
- **SEO-Optimized**: Custom, page-specific dynamic document titles and meta descriptions for search engine indexing.

---

## 🛠️ Technology Stack

- **Frontend Core**: React 18, TypeScript, Vite (Fast build tool)
- **Styling**: Tailwind CSS (Utility-first styling framework)
- **Animations**: Framer Motion (Smooth page transitions and micro-animations)
- **Icons**: Lucide React (Clean, minimal outline icon system)

## 🚀 Running Locally

Follow these steps to set up and run the project locally on your machine.

### Prerequisites

Make sure you have [Node.js](https://nodejs.org/) installed (v18+ recommended).

### Setup Instructions

1. **Clone the repository:**
   ```bash
   git clone https://github.com/syedsz-1519/Qalbiya_.git
   cd Qalbiya_
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```
   Open your browser and navigate to `http://localhost:5173`.

4. **Build for production:**
   ```bash
   npm run build
   ```
